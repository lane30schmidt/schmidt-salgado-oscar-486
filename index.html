These are the steps to start your repo and begin coding.

  • Download node.js version: 18.14.0 LTS: https://nodejs.org/en/
  • Create your own Repository on GitHub (.gitignore template ---> 'node')
  • Click the green 'Code' button in the repo and copy the HTTPS link
  • Open your work space in VsCode and open your git bash terminal and change directory to your work space.
  • Check that node is installed. If so, its time to clone your repo and begin coding. Remember to change directory into your newly cloned folder.
  • MAKE SURE YOU ARE TESTING YOUR CODE LOCALLY AS YOU ARE WORKING! (live server, xampp)




Time to 'Salute'! üíæüíæüíæ

Once you have finished coding, its time to send your work back to the repo!

  • First, save all your work and use the 'git status' command in your terminal to check which files need to be added.
  • Use the 'git add' command to add all the files that appeared in your git status response. Call 'git status' again to see your newly added files

    • -pro tip: 'git add .' will add all the files without having to type each one

  • Use the ' git commit -m " " ' command to create a commitment to the repo. '-m' represents your commit message you want displayed on the repo.
  • Finally, use the 'git push' command to push all of your work to the repo.

    • Refresh the page, and if you see an updated repo with your new commit message, you're all done!
